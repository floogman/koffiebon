Wat het is

Een klant koopt vooraf een aantal koppen koffie — bijvoorbeeld twaalf. Die staan op een digitale kaart die in de browser leeft en zich toont via een QR-code. Geen app, geen papier.

☕

Prepaid in koppen

De kaart ís een aantal koppen koffie, vooruit betaald. De eenheid is een kop, niet een bedrag.

📱

Leeft in de browser

Een PWA. Geen download, geen App Store, geen installatie.

🔳

Toont een QR

Elke kop wordt afgeboekt doordat de balie de QR scant.

QR in de PWA

Telefoon toont de code bij elke koffie.

Kaart kwijt of nieuwe telefoon? Eén herstel-link uit de bevestigingsmail laadt de kaart terug. Er ging nooit iets verloren — de kaart leeft op de server.

De QR is "dom". De server is de waarheid.

Roterende, eenmalige QR per koffie. De PWA vraagt elke keer een verse code aan de server. Een screenshot is meteen waardeloos.
Saldo staat centraal, niet op de telefoon. De balie boekt af op de server. Een kaart kan nóóit méér dan 12 keer.
Eén gebaar. Eerste scan = activeren. Elke volgende scan = −1 koffie. De balie hoeft niks te kiezen.
Herstelbaar. De PWA bewaart niets. Een link uit de e-mail laadt de kaart terug op elk toestel.

In het kort

☕
Een kaart is koppen koffie, vooraf betaald. Prijs en korting zijn losse knoppen.
📱
Een PWA toont een roterende QR. Geen app, geen papier, herstelbaar via e-mail.
🔒
De server houdt het saldo bij. Eenmalige codes maken fraude zinloos.
🧾
De balie activeert en boekt af met één scan. Betaling nu fysiek, later via Mollie.
